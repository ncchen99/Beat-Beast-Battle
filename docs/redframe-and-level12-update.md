# 紅框修復與第一二關邏輯更新

## 修改日期
2025-11-27 23:15

## 修改內容

### 1. 紅框顯示修復 ✓

**問題**：紅框 (`.red-frame`) 和金黃色框 (`.animal-frame`) 使用 `inset: -3px`，但父容器 `.animal-cell` 設定了 `overflow: hidden`，導致框被切掉而無法顯示。

**解決方案**：
- 更新 `src/index.css`，將 `.red-frame` 和 `.animal-frame` 的 `inset` 改為 `0`。
- 這樣框會顯示在格子內部邊緣，不會被切掉。

### 2. 第一關與第二關邏輯更新 ✓

**需求**：
- 開始後重複倒數 5 秒（自動循環）。
- 按鈕改為「A隊答錯」和「B隊答錯」，由裁判手動判定。
- 不需要每隊念完都點擊按鈕，只有在有人答錯時才點擊。

**修改詳情**：
- **自動循環倒數**：
  - 使用 `Timer` 組件配合 `key` 屬性。
  - 當 `onTimeout` 觸發時（5秒結束），自動切換 `currentTeam` 並更新 `timerKey` 以重置計時器。
  - 形成 A隊(5秒) -> B隊(5秒) -> A隊(5秒)... 的無限循環。

- **裁判按鈕**：
  - 移除原本的「通過/失敗」按鈕。
  - 新增「A隊答錯」按鈕：點擊後 B 隊獲勝，遊戲結束。
  - 新增「B隊答錯」按鈕：點擊後 A 隊獲勝，遊戲結束。
  - 新增「停止」按鈕：暫停遊戲但不計分。

**修改檔案**：
- `src/index.css`
- `src/levels/Level1.tsx`
- `src/levels/Level2.tsx`

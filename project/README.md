# 服學好好玩

一個五關團康互動網頁遊戲,採用可愛像素風格。

## 功能特色

- 🎮 五關挑戰:語言創意類 + 反應速度類
- 👥 A/B 隊對戰模式
- 🎨 像素風格美術設計
- ⏱️ BPM=91 節奏引擎
- 📱 響應式設計

## 技術棧

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (狀態管理)

## 開發指令

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 專案結構

```
src/
├── components/       # 可重用元件
│   ├── layout/      # 佈局元件 (Header, Footer)
│   └── game/        # 遊戲元件 (Timer, RhythmGrid)
├── levels/          # 各關卡頁面
├── store/           # Zustand 狀態管理
└── App.tsx          # 主要應用程式
```

## 遊戲說明

### 語言創意類
- **第一關**:照樣造句 (什麼來什麼去)
- **第二關**:創意接龍 (一...就...)

### 反應速度類
- **第三關**:說動物挑戰 (動畫版,可重複)
- **第四關**:進階挑戰 (真實照片,可重複)
- **第五關**:終極挑戰 (真實照片,不重複)

## 節奏遊戲規則

- **前 8 拍**:準備時間
- **中 8 拍**:遊戲時間 (跟著紅框念動物)
- **後 8 拍**:換人時間
- **總計**:24 拍循環
